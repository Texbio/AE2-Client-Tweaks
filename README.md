# AE2 Client Tweaks

A client-side Forge mod for Minecraft 1.20.1 that adds quality-of-life tweaks to [Applied Energistics 2](https://www.curseforge.com/minecraft/mc-mods/applied-energistics-2). 
</br>Currently only supports 1.20.1 forge.
</br>\- Created with Claude.

**Download:** [CurseForge](https://www.curseforge.com/minecraft/mc-mods/ae2-client-tweaks)

## Features

- **Shift-click patterns into Molecular Assemblers** — Shift-click encoded patterns from your inventory in the Pattern Access Terminal to insert them into the first available Molecular Assembler slot. Supports rapid shift-clicking.

  > **Note:** AE2 added shift-click insertion natively in 1.21.1, but it was never backported to 1.20.1. This mod brings that functionality to 1.20.1.

- **Processing patterns** — Processing patterns are automatically blocked from being shift-clicked into Molecular Assemblers

- **Craftable view pattern lock** — When a regular ME Terminal is set to "View: Craftable", encoded patterns are prevented from being shift-clicked into the network (useful if you have a sub-terminal where you view all crafting recipes).

## Compatibility

- Client-side only — no server installation required
- Works with **AE2 Wireless Terminal Library** (wireless pattern access terminal)
- Works with **Extended AE** and should work with other AE2 addons

## Requirements

- Minecraft 1.20.1
- Forge 47+
- Applied Energistics 2
